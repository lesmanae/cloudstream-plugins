# 🎬 Lesmanae Cloudstream Plugins

Plugin CloudStream buatan sendiri untuk konten hiburan berbahasa Indonesia.

---

## 📦 Cara Install

1. Buka **CloudStream** → **Settings** → **Extensions**
2. Klik tombol **+** → **Add Repository**
3. Masukkan URL berikut:
https://raw.githubusercontent.com/lesmanae/cloudstream-plugins/master/repo.json
---

## 🛠️ Tools & Teknologi

- Kotlin
- Cloudstream3 API
- Cloudflare Workers
- GitHub Actions

## 🌐 API

Plugin ini menggunakan API dari **Sonzai X** dengan mirror pribadi via Cloudflare Workers.

| API | URL |
|-----|-----|
| Sonzai X (Original) | [api.sonzaix.indevs.in](https://api.sonzaix.indevs.in) |
| Mirror (Cloudflare Workers) | [proxy-api.o69o.qzz.io](https://proxy-api.o69o.qzz.io) |
